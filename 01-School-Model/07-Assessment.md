| Method | Primary Purpose | Domain Focus |
| ----- | ----- | ----- |
| Portfolio-Based Assessment | Continuous documentation of student work and co-defined goals | All four domains |
| The Learning Record | Teacher observation, parent input, student self-report, AI data | Element \+ Whole Child |
| Performance-Based Assessment | Demonstrates applied knowledge through authentic tasks | Knowledge \+ Competencies |
| Competency-Based Assessment | Measures mastery of predefined skills, not time served | Timeless Competencies |
