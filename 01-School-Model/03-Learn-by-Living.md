## 3.1 What "Learn by Living" Means {#3.1-what-"learn-by-living"-means}

## *Why this is the way humans have always learned best*

* **Opening analogy** — one concrete, universal example of how humans learn naturally (e.g. learning to cook, learning a language abroad). No classroom. No syllabus. Real situation, real feedback, real progress.
* **The general principle** — humans learn by doing, not by being told. This is not a philosophy. It is how learning works.
* **The three shifts** — shown as a comparison table:

|  | How Humans Learn Naturally | Traditional School | theNORMALschool |
| ----- | ----- | ----- | ----- |
| **Where learning happens** | Inside real situations | Classroom first, application later — if ever | Inside real work, real contexts, real problems |
| **What students produce** | Something real and useful | Answers for the teacher — tests, essays, assignments | Real outputs for real audiences — prototypes, reports, campaigns |
| **Who drives the learning** | The learner — pulled by genuine need or curiosity | The teacher — push delivery to all students at the same pace | The student — pulled by their own project, curiosity, and need to know |

* **Closing line** — theNORMALschool is not a new model. It is the oldest model — built deliberately, at scale.

## 3.5 The Learning Flow: From First Encounter to Completed Cycle {#3.5-the-learning-flow:-from-first-encounter-to-completed-cycle}

## *How a complete element cycle runs — start to finish*

* Entry: Deep Assessment seeds the first project domain
* Hook: Entry event sparks curiosity and surfaces the driving question
* Inquiry: Student identifies what they need to know
* Building: Sustained work — research, prototyping, creating
* Critique: Peer and expert feedback; revision
* Public product: Real output for a real audience
* Reflection: What the student learned about the work — and about themselves
* Hypothesis update: Student Hypothesis and Personal Pathway revised
